# 🧠 NeuroScan AI

An AI-powered web application for **early Alzheimer's Disease detection** using **EEG signals**, a **Supervised Autoencoder**, and **Machine Learning**.

---

## 📌 Overview

NeuroScan AI helps classify EEG recordings into:

- 🟢 Healthy Control
- 🔴 Alzheimer's Disease

The system extracts meaningful EEG features using a **Supervised Autoencoder** and predicts the patient's condition using multiple machine learning models.

---

## 🚀 Features

- Upload EEG (.set) files
- Automatic EEG preprocessing
- Supervised Autoencoder feature extraction
- Multiple Machine Learning models
- Confidence score prediction
- Interactive React dashboard
- Model comparison page
- Confusion matrices
- Performance metrics
